# Meal App
A Flutter project that helps you to explore recipes then find your meals as you like, for example, lactose-free meals, and you can also add your favorite meals.
 
 # Specification
 <ul>
 <li>Android studio</li>
 <li><b>Dart</b> language</li>
 </ul>
